## Philosophy

QuickStock is designed around:
- fast kitchen-friendly UX
- shared household workflows
- low-friction inventory tracking
- practical meal planning

The project intentionally prioritizes:
- simplicity
- iteration speed
- MVP-first development

before:
- AI
- automation
- complex integrations
